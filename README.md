# TaskFlow

TaskFlow é um gerenciador de tarefas com interface moderna e clean, focado em organização e produtividade.

## ✨ Funcionalidades

- Cadastro e login de usuário
- Criação, edição e exclusão de tarefas
- Filtro por status (todas, pendentes e concluídas)
- Busca em tempo real
- Dashboard com estatísticas
- Interface responsiva e modo escuro

---

## 🛠 Tecnologias

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Banco de dados: SQLite
- Autenticação: bcryptjs

---

## 🚀 Como executar

```bash
git clone https://github.com/nulldev7x/taskflow.git
cd taskflow
npm install
npm start
